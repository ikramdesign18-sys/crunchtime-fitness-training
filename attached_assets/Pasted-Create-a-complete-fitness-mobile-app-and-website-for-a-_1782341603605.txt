Create a complete fitness mobile app and website for a fitness business.

App name: Crunchtime Fitness Training

Build the mobile app using React Native Expo with TypeScript. Create a clean, modern, beautiful, premium, user-friendly fitness app with perfect spacing, clean layout, smooth navigation, and professional light/dark theme support.

Use the uploaded logo/brand screenshot as inspiration. The logo color should be used as the primary brand color. Do not copy any ugly default UI. Make the app look polished, modern, and client-ready.

Create both sides inside one mobile app using role-based authentication:

1. User/Consumer side
2. Trainer/Admin side

Use one login page. After login, check user role:

* If role is user, open user dashboard
* If role is trainer/admin, open trainer dashboard

Normal signup users should automatically get role = user.
Trainer/admin account should be created manually in the database.

Use Supabase for authentication, database, and file storage. If Supabase keys are not added yet, create placeholder config and clear setup instructions in the code comments.
